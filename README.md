วิธีการเล่น

1.ว่างเดิมพัน

2.เลือกแทงสูงต่ำ

3.กดปุ่มเขย่า

4.ลุ้นว่าออกสูงหรือต่ำ

5.เล่นเสร็จให้กด CLEAR
